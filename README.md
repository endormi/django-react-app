# Django-React-App

**Backend**: Django (**2.1.7**)

**Frontend**: React.js (**3.0.1**)

**NGINX**: NGINX proxy with Docker

## Running the server

You must have [docker](https://www.docker.com/) and [docker compose](https://docs.docker.com/compose/) installed.

```
docker-compose up
```

Build

```
docker-compose build
```
